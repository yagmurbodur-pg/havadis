# Kod ajanına ücretsiz internet erişimi: wigolo

wigolo, yapay zekâ kod ajanlarına internette arama, sayfa çekme ve tarama yeteneği ekleyen bir araç. Her şey kendi bilgisayarında çalışıyor: API anahtarı yok, bulut yok, sorgu başına ücret yok. Ajanlara MCP (araçları yapay zekâya bağlayan standart protokol) üzerinden bağlanıyor. Şu an herkese açık beta aşamasında ve GitHub'da hızla yükseliyor.

**Neden önemli?** Bugün dene: Claude Code gibi bir ajan kullanıyorsan, wigolo ile ona ücretsiz web araması ekleyebilirsin.
